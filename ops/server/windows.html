# Windows 服务器运维

# Windows Server 2019

# Docker 安装

下载：Docker Desktop for Windows - x86_64 (opens new window)

# 基于 WSL2

  • WSL version 1.1.3.0 or later.
  • Windows 11 64-bit: Home or Pro version 22H2 or higher, or Enterprise or Education version 22H2 or higher.
  • Windows 10 64-bit: Minimum required is Home or Pro 22H2 (build 19045) or higher, or Enterprise or Education 22H2 (build 19045) or higher.
  • Turn on the WSL 2 feature on Windows. For detailed instructions, refer to the Microsoft documentation (opens new window).
  • The following hardware prerequisites are required to successfully run WSL 2 on Windows 10 or Windows 11:
    • 64-bit processor with Second Level Address Translation (SLAT) (opens new window)
    • 4GB system RAM
    • Enable hardware virtualization in BIOS. For more information, see Virtualization (opens new window).

For more information on setting up WSL 2 with Docker Desktop, see WSL (opens new window).

# 基于 Hyper-V

  • Windows 11 64-bit: Home or Pro version 22H2 or higher, or Enterprise or Education version 22H2 or higher.
  • Windows 10 64-bit: Minimum required is Home or Pro 22H2 (build 19045) or higher, or Enterprise or Education 22H2 (build 19045) or higher.
  • Turn on Hyper-V and Containers Windows features.
  • The following hardware prerequisites are required to successfully run Client Hyper-V on Windows 10:
    • 64 bit processor with Second Level Address Translation (SLAT) (opens new window)
    • 4GB system RAM
    • Turn on BIOS-level hardware virtualization support in the BIOS settings. For more information, see Virtualization (opens new window).
